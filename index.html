checklist

Checklists contain distinct taxa ordered by frequency of occurrence and are built directly from iDigBio's Darwin Core Archive (~15M specimen records). If the checklist has not been calculated before, this may take several minutes. See Checklist Processing Dashboard for checklist calculation status. Supports comma separated list of focal taxa (e.g. "Aves,Insecta").

occurrences

Occurrences are retrieved directly from the GBIF API. This list may contain duplicates and is not ordered in any way. Only supports single focal taxa (e.g. "Aves", but not "Aves,Insecta").
